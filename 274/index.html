Title: Readify Wikipedia bookmarklet
Year: 2023
Medium: Javascript bookmarklet
Elevator pitch: Makes it much easier to read Wikipedia without distractions. All links are hidden (turned to black, no underline), sidebar tables and navigation hidden, body text centered and given a new max-width. Install by placing on browser bookmarks bar.
