ZineMaster2054 Documentation

Note: Click on any page you've scrolled to and it will replace that page with a different layout and selection of images

  • File Selection - Click to add a single file or select multiple files at once. They will be added as potential source images for your zine and your pages will be scrambled and remixed using these new images.

  • Color Picker - This color picker is used to set the background color block behind the zine title and artist name

  • Change Title - Change the default title of the zine. It attempts to scale the size to the title's length, within reason. Deleting the text will reset the name to ZineMaster2054. Want a blank title? Use a space

  • Add Artist's Name - Use this to add a credit line to the front of your zine

  • Number of Pages - Enter a number greater than 0 and less than 255. Each time you change this your zine pages and images will be scrambled and remixed using the new page length.

  • Print - Prints out an image file of the entire length of your zine, removing the buttons from the printed version

  • Help - It got you here.

ZineMaster2054 is by Lee Tusman. The initial prototype was created in a day during regular code sketch time, though the idea had been floating in my head for years. I'd love to see anything you make. And please send feedback and suggestions to me or check out the code.
